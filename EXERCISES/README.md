# EXERCISES

Эта папка для самостоятельных упражнений по архитектуре.

Цель: научиться самому раскладывать игровую механику на маленькие классы, понимать где нужен Zenject, где достаточно `SerializeField`, где уместна фабрика, и как не нарушать SOLID.

Начни с файла:

```text
01_Gates_DebugLog_Architecture_Practice.md
```

Упражнение специально можно сделать только через `Debug.Log`, без префабов, моделей, Addressables и сложной сцены. Так легче увидеть именно логику архитектуры.
